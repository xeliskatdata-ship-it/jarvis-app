# Politique de sécurité

## Signaler une vulnérabilité

Merci de **ne pas** ouvrir d'issue publique pour une faille de sécurité.
Utilise le canal privé GitHub : onglet **Security → Report a vulnerability**,
ou contacte le mainteneur en direct.

Réponse sous ~7 jours. Projet personnel, sans SLA de production.
